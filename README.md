**Description:**
This project will be built with the goal of assisting a user track/meet diet and health goals through the tracking of meals and their macronutrient statistics. The main source of data will be meals, which will consist of attributes like the meal name, it's numeric macronutrient values (protein, carbs and fats), the time the meal was eaten and the type of meal (breakfast, lunch, dinner, snack). These meals will be used to compile summary statistics that the user can view based on a time window or the meal type. These data points could then be used to track a users progress towards inputted macronutrient goals and larger health goals (building muscle, losing weight, etc.). The health goals could then be refined using inputted health information from the user (weight, height, etc.)
